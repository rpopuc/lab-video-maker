# Credentials Format

## Algorithmia

File: `algorithmia.json`
```
{
    "apiKey": "API KEY"
}
```

## Watson Natural Language Understanding

File: `watson-nlu.json`
```
{
    "apikey": "",
    "iam_apikey_description": "<api description>",
    "iam_apikey_name": "",
    "iam_role_crn": "",
    "iam_serviceid_crn": "",
    "url": ""
}
```
